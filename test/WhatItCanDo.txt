Spaces in between keywords
Soft line breaks (comma)
Scoping (since nested if-else is not implemented, not sure if dynamic or static)
-------------------------
Functions (HOW DUZ I, IF YOU SAY SO, FOUND YR, GTFO)
	HOW DUZ I [function name] [arg1] AN [arg2] ...- starts the function
	IF U SAY SO - ends the function
	FOUND YR [expression]- returns a value. Stored in IT if no variable will catch it
	GTFO - returns NOOB

	I IZ [arg1 [AN arg2 [AN ...]]] MKAY - calls the function
Typecasting (MAEK, IS NOW A)
	MAEK [expression] A [type]- changes an expression to a datatype
	[variable name] IS NOW A [type] - changes the a variable to a datatype
LOOPS
	IM IN YR <label> UPPIN/NERFIN YR <variableIdentifier> TIL/WILE <condition
	IM OUTTA YR <label>
-----------------------
HAI and KTHXBYE are not required to be at the start or at the end of the code
